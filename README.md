# Liri
- Liri is an ap that lets you search for a song, movie, or see if there is a band in town!


# What we used.

  - JavaScript
  - Node.js
  - Axios Node Package
  - Node Spotify API
  - Bands In Town API
  - OMDB API
  
# How to use

  - Get an API key for each of the APIs listed above
  - Run npm install
  - Put your api keys inside key.js
